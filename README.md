# Results Summary Component

This is a solution to the **Results Summary Component** challenge from Frontend Mentor.

The project is built using **HTML and CSS** and focuses on creating a clean, responsive layout that adapts well to different screen sizes.

## Features

- Responsive design (mobile & desktop)
- Clean and semantic HTML structure
- Styled using modern CSS (Flexbox & Grid)
- Hover states for interactive elements

## Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid

## Preview

Design files provided by Frontend Mentor were used as reference.
